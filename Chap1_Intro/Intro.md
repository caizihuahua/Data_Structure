## Introduction to Data Structure

---

### Defination

data type :
- data object (data element)
    - data item (data filed)
- data structure (the relationship between the element)

### Abstract Data Type (ADT)

```
ADT Triple{
    data_object
    data_relationship
    basic function:
        InitTriple (&T, v1, v2, v3)
        DestroyTriple (&T)
        Get (T, i, &e)
        Put (&T, i, e)
        IsAscending (T)
        IsDescending (T)
        Max (T, &e)
        Min (T, &e)
}ADT Triple
```
